# Trader Journal — pre-registered

Pre-registered public trade log. Every entry is written **before** the trade
is submitted at the broker. Git commit timestamp = tamper-evident proof of
pre-registration; monthly broker exports prove the fill sequence.

**This is not investment advice.** Published as a discipline mechanism and a
public track record.

## Design rationale, one paragraph

Hindsight bias is symmetric: after a win, memory reconstructs the pre-trade
state as cleaner than it was; after a loss, memory manufactures a "warning
voice" that may never have existed. Post-hoc journals inherit both
distortions. Kahneman & Klein (2009), *"Conditions for Intuitive
Expertise"*, argue expert intuition can only be validated when captured
before the outcome arrives. This journal captures the "gut read" and any
"ignored warning voice" pre-trade so they can be scored against outcomes
without hindsight contamination. Over 50–100 entries, patterns become
falsifiable: which "I knew it" reads actually predict, and which warnings
actually matter.
